정렬은 문자열과 요소로 나누어서 처리

        [문자열 정렬]
        text-start : 왼쪽정렬
        text-center : 가운데정렬
        text-end : 오른쪽정렬
        text-justify : 양쪽정렬

        [요소(태그) 정렬]
        div내에 d-flex 선언 후 사용
        justify-content-start : 요소를 왼쪾 정렬
        justify-content-center : 요소를 가운데 정렬
        justify-content-end : 요소를 오른쪽 정렬
        justify-content-btween : 요소를 양족 정렬
        justify-content-around : 요소를 간격 균등
        justify-content-evenly : 요소를 최대 간격

        align-items-start : 요소를 위쪽에 정렬
        align-items-center : 요소를 가운데 정렬
        align-items-end : 요소를 아래쪽에 정렬
        align-items-stretch : 요소들이 높이를 균등
