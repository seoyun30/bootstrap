해당 요소를 기준으로 해서 바깥쪽백여백(margin), 안쪽여백(pading)으로 구성
      크기는 1~5사이의 값

      [바깥쪽여백]
      m-크기 : 상하좌우의 여백을 적용
      mt-크기 : 위쪽 적용(top)
      mb-크기 : 아래쪽 적용(bottom)
      ml-크기 : 왼쪽 적용(left)
      mr-크기 : 오른쪽 적용(right)
      mx-크기 : 좌우 적용
      my-크기 : 상하 적용

      [안쪽여백]
      p-크기 : 상하좌우의 여백을 적용
      pt-크기 : 위쪽 적용(top)
      pb-크기 : 아래쪽 적용(bottom)
      pl-크기 : 왼쪽 적용(left)
      pr-크기 : 오른쪽 적용(right)
      px-크기 : 좌우 적용
      py-크기 : 상하 적용

      각 요소별 간격 또는 크기 지정
      여백시 내용이 가운데 존재하면 여백이 적용
      여백시 내용이 상단에 존재하면 여백이 적용되지 않은 상태
